我对搜索引擎的一些经验

Introduction

写完我对整理笔记的工具的一些经验，再来写写我对搜索引擎的一些心得。

受到如何正确提问这篇帖子的影响，我本人对提问一概来者不拒（尤其是高质量的问题），但对于可以通过STFW（Search The Fucking Web）解决的问题则一直不是很耐烦——你谷歌一下第一条就能查到的东西，还跑来问我，对你对我都是浪费时间精力，何苦呢？然而相处下来，发现并不是所有人都能很好地利用这个fucking web，对于搜索引擎的使用意识和使用方法都存在缺陷，并非是他们不想用，而是他们真的不会利用。

在此献丑一番，讲点我对搜索引擎的经验。除了我们生活中通常定义为“搜索引擎”的那些工具之外，还有一些软件在特定用途上，因为其内容的独家性和特殊性，使用效果媲美甚至超过搜索引擎。

谷歌

如果你之前使用的是必应，那么我认为谷歌可以提升你50%的使用体验；如果你之前使用的是百度，那么谷歌可以提升250%的体验。

使用谷歌搜索的优点包括

  • 80%的问题可以通过谷歌解决
  • 外文搜索体验远超百度
  • 搜索内容包括youtube、Stack Overflow、维基百科等内容，范围远超百度
  • 广告少，内容准确，完爆百度

缺点嘛，就是众所周知的原因。而且如果你不常访问，那么谷歌可能会经常因为认为你是机器人而让你进行图形码验证（而且我个人觉得，这个验证还挺难的orz，经常做错）。对此网上有一些解决方案，我的方法是直接注册一个谷歌账号，这样它就很少会判定机器人了。

小红书

小红书不愧为“生活经验的百科书”。就我观察，小红书近年来优质内容产出越来越多，尤其是在生活领域这一块，市面上无人能敌，集美们都抓紧用起来吧。

这些领域适合使用小红书搜索：

  • 一系列攻略（旅游、美食、医疗，甚至是怎么跟女孩子聊天，去哪里约会等等）

  • 一系列教程（烹饪、摄影、护肤、穿搭、发型、星座、健身、宠物，甚至是考研、保研、高考、四六级。我甚至用小红书查原神攻略（确信））

  • 资源分享

  • 寻求共鸣（xhs上会有很多集美遇到与你一样的烦恼与焦虑）

我自己就有一些亲身的经历。例如去年新冠的时候，我从怎么囤药，囤哪些药，到阳性时遇到症状怎么处理，都是通过xhs搜索的。除了可以得到官方的提示之外，还可以看遇到类似症状的集美是怎么处理，效果是怎么样的。后者的经验我觉得是其他平台上很难获得的。

再如前两天我和小宝出去玩，前一天晚上打算去上海图书馆，结果打开小程序发现往后一周的预约全被抢完了。本来以为没办法进去只能作罢，直到我试了试在xhs上查了下，一查就查到集美的攻略：

80 Hermione的书单📖发布了一篇小红书笔记，快来看吧！ 😆 fqEf4g2gt7lZ4JF 😆 http://xhslink.com/co0Kln，复制本条信息，打开【小红书】App查看精彩内容！

集美在帖子里指了一条路：通过预约展览等活动，也可以进入图书馆，并且只要完成签到，剩下的活动任君安排。果不其然，活动区还有近两百张票剩余。第二天我和小宝就靠着预约票进入了图书馆。咱就是说集美们，小红书真的是绝绝子！

微信

微信自带的搜索也有一定的独特作用，不过这独特倒不是体现在什么特点上，而单纯只是因为微信的体量大，却又不公开给别的搜索引擎罢了（）

微信的独特优势在于：

  • 微信独家的文章内容
  • 公众号，包括与之相关的一系列资源（四六级，数模，等等）

我自己的一个典型例子是用微信搜索下载历年的四六级真题。平日里经常能在各个公众号看到速成四六级的广告，那么逆向推理来看，四六级的资源在公众号也不会少。网上的许多所谓免费资源，也是关注xx公众号，乃至于分享pyq后才能获得。与其起去别的平台绕圈子，不如直接去微信溯源追踪。

微博

  • 时事新闻
  • 表情包（是的，很隐藏的用途，哈哈。但是确实有很多隐藏的表情包）
  • 原神超话签到3天领取签到码（确信）

百度

前文里我对百度已经进行了一番批判，如果要让我再客观一些地说，那么当且仅当你需要的内容范围在中国大陆内且语言为中文时，百度具有一定的优势。当然尽管如此，你也需要注意第一页可能都是广告的情况。

另外我经常使用百度app的拍照识图功能，这点我觉得还是做得挺不错的。

zlibrary

毋需多言，现在zlibrary也已以各种形式重新复活（想知道方法？使用搜索引擎吧！），查阅电子书这一块还是被zlibrary拿捏得死死的，无广告下载快（有梯子时），其他电子书厂商看了都叹息。

GitHub

程序员都熟悉这个平台，不过除了在上面获取各种代码外，GitHub里也有一些不错的资源，不过主要以程序员的需求为主，包括简历模板、面试经验等。我之前也在上面查询过数模的相关资料。

ChatGPT

临时想到最近大火的ChatGPT，也应该加入到讨论的列表中来。我自己也是在tb上买了个ChatGPT的号，只要挂上的梯子不是港澳台，基本上就能和ai直接对话。使用下来确实一番惊艳，可以感受到未来的趋势，不过现在仍然有许多改进之处（使用人数太多，回应缓慢，答案有时也不一定准确）。我个人认为，以下两个应用场景（同时也是新款bing搜索提供的特性）最能体现ChatGPT的风采

  • 知识百科。通过ai整合各个知识来源，往往可以使你更快get到想要获得的知识点

  • 写作助手，文案生成器。告诉他你的需求，他会把你想要的文案给你，即使无法直接用作初稿，稍稍润色一番也已经相当不错。

高级搜索

使用高级搜索可以让你的搜索更加精确。例如，在百度和谷歌里 使用英文引号表示精确查询，即必须有此内容，用减号表示不需要相关内容，使用site选择搜索来源等等。学过类似文献检索的uu应该都知道这些内容。

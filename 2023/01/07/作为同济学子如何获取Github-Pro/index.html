作为同济学子如何获取Github Pro

认证

受到友人启发，在其指导下完成了Github Pro的认证（截止到2023.1.7）。Github Pro因为一度被滥用，现在审查非常严格。现总结流程如下：

  1. 可以的话，尽量在学校或离学校不要太远。Github在预检查的时候会查看你的定位是否符合要求
  2. 使用Safari等浏览器（不推荐Google，原因后面解释）前往https://education.github.com/，选择`Join Global Campus`
  3. 选择一番（尽量用学校邮箱，更容易验证通过），即可进入认证的页面，Github会让你上传证件来证明。这方面Github限制比较严格，我是这么做的（like my friend）
  4. 拍摄你的录取通知书（记得尽量拍得平整，尽量高清，因为Github会进行OCR扫描）
  5. 认证。这里我强调一下为什么要用Safari等浏览器，主要是排除Chrome浏览器。因为我一开始用Chrome的时候，发现它只能选择用电脑摄像头拍摄，直到跟友人一交流，发现Safari还多出了一个上传本地选项，而Chrome..居然没有
  6. 选择Upload an image，上传你的录取通知书；Proof Type选择Dated School ID（大概这么称呼）
  7. 接下来可能还会报一些错。按着错误描述一个一个解决就好

接下来…我也不知道为什么，马上就通过了。只能说，同济学子用录取通知书获取Github Pro认证，绝对是可行的

URL

scheme://host.domain:port/path/filename

名称 功能 最常见
scheme 因特网服务的类型 http https(安全超文本传输协议) ftp file
host 域主机 默认www
domain 因特网域名 例如baidu.com
:port 主机上的端口号 默认80
path 服务器上的路径 如果省略 则文档必须位于网站根目录中
filename 文档/资源的名称

只能使用ASCII字符集！！

原来复制 Mac 文件路径可以这么简单

说出来有点丢人——我用 Mac 快四年了，直到今天才知道怎么快速复制一个文件夹的完整路径。

之前我的操作流程是这样的：

  1. 打开终端
  2. 把文件夹拖进去（或者手动 cd 过去）
  3. 输入 pwd，回车
  4. 用鼠标选中输出结果，复制

每次都要切到终端，敲几个命令，就为了拿一个路径。说实话我都习惯了，觉得这就是 Mac 的「正常操作」。毕竟 Windows 上好歹还有个地址栏可以直接复制，Mac 的 Finder 你点一下标题栏只是切换层级，没有那种可以直接复制的路径栏。

直到今天我问了一下 AI。

AI 告诉我：在 Finder 里选中文件夹，按住 ⌥ Option 键再右键点击，菜单里的「复制」会变成「将 xxx 拷贝为路径名称」。

就这么简单。

那一刻我沉默了几秒。四年。四年里我敲了多少次 pwd？

其实还有另一个更快的方式：选中文件夹后直接按 ⌥ Option + ⌘ Command + C，路径就直接进剪贴板了，连右键都不用。

两个方法本质一样，看你习惯键盘还是鼠标。我个人更喜欢快捷键版，因为少一步操作，而且有一种「这才是 Mac 该有的样子」的感觉。

如果你也跟我一样，一直用终端手动 pwd 复制路径——这篇文章就是为你写的。试试看，真的会觉得自己之前好憨。
